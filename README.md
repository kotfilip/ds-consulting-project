# Projekt: Analiza śmiertelności w Polsce (2015–2023)

Repozytorium do projektu z przedmiotu 'Data Science – Consulting Approach'.

## Struktura
- `src/` – kod źródłowy
- `data/` – dane wejściowe i przetworzone
- `notebooks/` – analiza eksploracyjna, testy modeli

## Uruchomienie
```bash
python main.py
```
